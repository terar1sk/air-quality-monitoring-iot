# 🌍 IoT Air Quality Monitoring System

Engineering project developed using **Raspberry Pi and Cytron Robo platform** as part of a bachelor thesis focused on collecting and processing environmental measurement data.

---
## 🚀 Project Overview

This project implements an **IoT-based air quality monitoring system**
designed to measure environmental parameters and transmit collected data
to a database for further analysis and visualization.

The system combines embedded hardware, sensors, and software components
to enable continuous environmental monitoring.

---
## ⚙️ System Features

- environmental data measurement
- sensor integration via microcontroller
- automated data collection
- database data transmission
- real-time monitoring capability
- modular hardware architecture

---
## 🧩 Hardware Components

- Raspberry Pi
- Cytron Robo controller board
- environmental sensors
- network connectivity module

---
## 🔄 System Workflow

1. Sensors collect environmental measurements.
2. Data is processed by the controller.
3. Raspberry Pi aggregates measurement data.
4. Data is transmitted to a database server.
5. Measurements can be analyzed and visualized.
